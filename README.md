# Web back end

This repository contains projects and exercises for the **third trimester** of Holberton School's first-year curriculum. The focus is on backend development concepts, Python, JavaScript, and NoSQL databases.

## Topics Covered

### Python
- Variable annotations and type hints  
- Pagination techniques  
- Asynchronous programming (`async/await`)

### NoSQL
- Basic operations and data modeling

### JavaScript
- Object-oriented programming with classes  
- Data manipulation and processing  
- Promises and asynchronous handling
